6. Identifying Sequence of Activities

Now, let us consider how a web browser and a web server communicate. A user types in a URL in the browser, and views the page. But something more happens under the hood to display a page.

When a user types in a URL and clicks on the "Go" button, the browser sends a HTTP Request to the concerned web server.

When the web server receives a HTTP Request from any client, it sends back a HTTP Response to that client.

On receiving a HTTP Response, web browser displays(renders) the requested page.

Thus we have identified the two broad activities that take place when we try to view a page on a web browser.
