Class Diagrams and Sequence Diagrams

We will learn how to draw class diagrams and sequence diagrams to model a system from a given problem statement. We will follow these steps:

  • 1. Represent classes with their state and behaviour
  • 2. Association between identified classes
  • 3. Aggregation of classes
  • 4. Composition of classes
  • 5. Inheritance of classes
  • 6. Identifying sequence of activities
  • 7. Drawing sequence diagrams
